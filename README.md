## 1. 프로젝트 개요
- 목적: 사용자의 접근을 감지하여 물리적 접촉 없이 쓰레기통 뚜껑을 자동으로 열고 닫음으로써 위생과 편의성을 증대

- 핵심 기술: 초음파 거리 측정(음파의 왕복 시간을 이용한 거리 계산 $$Distance = \frac{Time \times Velocity}{2}$$
    - PWM 제어: 서보 모터의 각도를 정밀하게 제어하여 개폐 동작 구현
    - 상태 제어: 감지 → 열림 → 유지 → 닫힘

## 3. 관련 자료
- **거리 측정 및 객체 감지 이론**
    - 초음파 센서(HC-SR04)의 원리
        - 트리거 핀을 통한 $10\mu s$ 이상의 펄스 발생 및 초음파 송출
        - 에코핀의 High 유지 시간을 통한 거리 환산 공식
        - 상온 기준 음속 데이터 및 온도 변화에 따른 음속 보정 수식

- **하드웨어 제어 및 통신 기술**
    - ESP32 핀 맵 및 PWM 제어
        - ESP32의 LEDC기능을 이용한 서보 모터의 정밀 각도 제어
        - 서보 모터의 동작 범위와 뚜껑 개폐 각도 간의 기계간 상관관계
    - 전원 공급 체계
        - ESP32 구동 전압(3.3V)와 모터 구동 전합(5V)분리 및 공통 접지 설

- **사용자 경험 및 인터페이스 설계**
    - 반응 속도 최적화
        - 사용자 접근 시 지연 시간 최소화 로직
        - 불필요한 감지로 인한 오작동 방지 설계
